# aws-installation

Prerequisites
SERVER
OS = CentOS7

Size = 4GB / 2 CPUs

Storage = 80GB

Installation
1. Install git
yum install git -y

2. Clone the repository
git clone https://github.com/azaa1/awx-installation.git

3. Change Directory to Repository
cd awx-installation

4. Run installation
bash install.sh

5. Access Tower
Open your browser, enter the public IP of the SERVER

6. Log In
user: admin

password: password
